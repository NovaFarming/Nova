# Allocation Bands

Nova optimizes yield after drag and rebalance friction, so it helps to define broad bands instead of pretending every basis point needs action.

## Core band

Use this when net APR is strong, IL drag is controlled, and utilization is stable. This is the default capital posture.

## Watch band

Use this when raw yield still looks attractive but one of the supporting inputs is weakening, such as liquidity, fee retention, or stress behavior.

## Exit band

Move here when the headline APR only survives on assumptions the engine no longer trusts, especially if friction is rising at the same time.

## Rule of thumb

The switch between bands should come from durable net improvement, not cosmetic rank changes between nearly equal options.
