# Nova Runbook

## What Nova Is For

Nova helps the operator allocate Solana capital into cleaner net carry after drag, utilization stress, and rebalance friction.

## Daily Operator Loop

1. Run `bun run dev`.
2. Read the allocation board for the top-ranked routes.
3. Confirm net APR, exit depth, and concentration are still acceptable.
4. Rotate only when the edge still survives friction.

## What Gets Promoted

- clean carry after drag
- good reward exit depth
- manageable utilization and protocol concentration

## What Gets Demoted

- emissions-heavy routes with weak exits
- high-utilization books
- rebalances that do not clear real costs
